# components/sections

Composable page sections — larger than UI atoms, smaller than full pages.

Examples: HeroWithMockup, FamiliarMoments, LifeAreasBento, ProductProofStrip, MorningStory, PricingCards, DownloadBand, LegalProse.

Shared across pages when useful; page-specific sections may live here with a page prefix (e.g. `home/HeroSection.jsx`).
