# utils

Pure helpers and static configuration — no JSX, no React imports.

| Module | Purpose |
|--------|---------|
| `constants.js` | Site name, URLs, contact, breakpoints |
| `routes.js` | Path builder helpers (`solutionPath('money')`) |
| `format.js` | ₹, phone (+91), date formatting |
| `content/` | (Future) Normalised copy from `docs/website/` |
